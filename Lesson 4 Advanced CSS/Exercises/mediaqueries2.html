Responsive Navbar with Theme Toggle

This example shows a responsive navbar that changes layout at 1024px. On larger screens, the navigation links display in a horizontal row. On smaller screens, they stack vertically behind a hamburger menu.

The page also includes a theme toggle with Light, Dark, and System options - all implemented with CSS only!

How It Works

This implementation uses CSS-only techniques:

  • Hidden checkbox inputs control the mobile menu toggle
  • CSS variables provide theming capabilities
  • Media queries handle the responsive breakpoints
  • The System theme uses the prefers-color-scheme media query
